# ProductManager

ProductManager es una aplicación móvil Android desarrollada con:
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Arquitectura MVVM

## Funcionalidades

✔ Registro de usuario  
✔ Inicio de sesión  
✔ CRUD de productos (Crear, Leer, Actualizar, Eliminar)  
✔ Datos en tiempo real usando Firestore  
✔ UI moderna en Jetpack Compose  
✔ Un usuario solo ve sus propios productos  

## Estructura del Proyecto

- `/data/model` → Modelos de datos
- `/data/repository` → Firestore y Auth
- `/viewmodel` → AuthViewModel y ProductViewModel
- `/ui/screens` → Pantallas Compose
- `/ui/components` → Componentes reutilizables

## Requisitos

- Android Studio Flamingo o superior
- Firebase configurado (Auth + Firestore)
- Conexión a internet
